---

## Features
### 1. User-Friendly GUI**
- Interactive and simple dashboard  
- Options for Inventory, Sales, Add Fruit, and Reports  
- Easy navigation for shopkeepers  
### 2. Inventory Management**
- Add, update, remove fruits  
- Track stock levels  
- Shows Low Stock / Out of Stock alerts  
- All data stored using **File I/O**  
### 3. Sales & Billing System**
- Select fruits and quantities  
- Auto calculates total price  
- Applies discounts  
- Generates invoice/receipt  
- Saves every sale in a **Sales Log file**
### 4. Discounts & Promotions**
- Auto discount for bulk purchase  
- Seasonal discount support  
- Accurate billing with applied rules  
### 5. Reports Module**
- Generate sales reports  
- View daily/monthly transaction logs  
- Helps in business analysis  
### 6. Exception Handling**
- Custom exception: **OutOfStockException**
- Prevents invalid purchase  
- Displays alert and stops process safely
